# QR Code Generator Plus

A comprehensive QR code generator built with Vue.js supporting multiple data types. Generate QR codes for various purposes with an intuitive interface.

## Features
- Generate QR codes for 10 different data types
- Real-time preview
- Download QR codes as PNG
- Responsive design
- Clean and simple UI

## Supported QR Code Types
1. Text (Free Text)
2. URL (Website Link)
3. Email Address
4. Phone Number
5. SMS
6. V-Card (Contact Card)
7. Wi-Fi Login
8. Location (Coordinates)
9. Event (Calendar Item)
10. Payment Information
